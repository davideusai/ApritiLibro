#Studio sui soprannomi del paese di Sestu (092074 - Italia)

##File Dataset
-Nome file : Allomingius_Dataset
Per una correta pronuncia si indica tra parentesi la n aspirata
Il formato di distribuzione  è CSV, separatore punto-virgola
Il dataset contiene un header
Una soprannome per record
Se più cognomi per un soprannome, questi sono separati dal carattere /

##File Tassonomia
Nome file : Allomingius_Tassonomia
Il formato di distribuzione è CSV, separatore punto-virgola
La nomenclatura contente la tassonomia contiente un header
